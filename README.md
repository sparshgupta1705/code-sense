# CodeSense AI

An AI-powered coding assistant built without using any external AI API.

## Features

- User Login
- Online C++ Compiler
- AI Code Review
- Time Complexity Prediction
- Automatic Test Case Generation
- Suggest Code Improvements
- Progress Dashboard

## Tech Stack

Frontend:
- React
- TailwindCSS

Backend:
- NodeJS
- Express
- MongoDB

Machine Learning:
- Python
- Scikit-learn
- Transformers (optional later)

Languages Supported

- C++
